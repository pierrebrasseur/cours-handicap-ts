    Mise à disposition

    Les PDFs ne sont pas hébergés dans ce dépôt par défaut. Pour les rendre téléchargeables sur le site déployé :

    1. Créer le dossier docs/pdfs/ dans le site
    2. Y copier les PDFs depuis Desktop/Audition_HETS_309/
    3. Reconstruire avec mkdocs build

    Commande de copie : 

    mkdir -p docs/pdfs
cp ../*.pdf docs/pdfs/
cp ../*.pptx docs/pdfs/
